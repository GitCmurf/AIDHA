## Purpose

The AIDHA source code is licensed under the Apache License 2.0. That license covers copyright in
the code. It does not grant permission to use project names, logos, or other brand identifiers in a
way that suggests endorsement, official status, or confusing association with the project owner.

## Project Name

"AIDHA" is the project name used by Colin Farmer / GitCmurf for this repository and related project
materials.

You may use the name "AIDHA" to:

- refer accurately to this project;
- describe compatibility with this project;
- link to or cite this repository;
- identify that your work is based on or derived from this project, when that statement is true.

You may not use the name "AIDHA" to:

- imply that an independent project, product, service, package, or organization is official;
- imply endorsement by Colin Farmer / GitCmurf;
- present modified versions as the original AIDHA project;
- use a confusingly similar name for an unrelated public project.

## Forks and Derivatives

Forks and derivative works should make their relationship to AIDHA clear. Prefer wording such as
"based on AIDHA" or "compatible with AIDHA" rather than names that imply official project status.

## Formal Registration

No formal trademark registration decision is recorded in this document. Formal registration can be
revisited if the project develops a public user or contributor community.
